# AORTAX VISION — Abo Wateen Studio

AORTAX VISION is a next-generation AI image studio focused on creative, high-fidelity visuals — now with Arabic-first UX.

**Features**
- Generate ultra-realistic images (Pollinations) — no watermark
- Prompt enhancer, styles, seed control
- Dark/Light mode + Arabic/English
- Local gallery (save & download)
- Text-to-speech (TTS) for descriptions
- PWA-ready (installable on mobile)

**Install / Run**
1. Put files in repository root (index.html, manifest.json, service-worker.js, icons).
2. Enable GitHub Pages (Settings → Pages → Source: main / (root)).
3. Open the published URL and use the app.
